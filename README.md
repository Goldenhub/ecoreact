# ECOREACT

## About Ecoreact

Ecoreact is a platform to search for and discover tools in the React Ecosystem.
It is updated regularly and requests are also taken from  the public.

## Built with

- Laravel
- Inertiajs
- ReactJS
- TailwindCSS
- Algolia search
